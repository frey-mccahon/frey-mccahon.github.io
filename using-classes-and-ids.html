Using Classes and ID Selectors


Classes and ID selectors are the stitches between your HTML and CSS. By adding one of these slectors to an HTML element, you link it to an atribute in your CSS style sheet. Here's an example of them in use and the syntax used.


Although both of these selectors can apply an atribute to an element, the ID selector can only be used to select one single element in an HTML page. Class selectors on the other hand can be applied to more than one element, because of this they are more useful and much frequesntly used.


Even if you are only selecting one element to style, if you think you may want to reuse the style on something else in future you should use a Class selector.
